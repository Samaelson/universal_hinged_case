# universal_hinged_case
Creates a 3D hinged case arround object to be used for outdoor wall mount

![Example of 3D hinged case for object dimensions 100x200x40 in Openscad.]([https://github.com/Samaelson/hinged_case_for_100x200x40_object.png](https://github.com/Samaelson/universal_hinged_case/blob/main/hinged_case_for_100x200x40_object.png))


The Belfry OpenScad Library, v2 is used, so needs to be installed (https://github.com/BelfrySCAD/BOSL2)
